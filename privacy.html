Privacy Policy

Last updated: July 19, 2026

The short version: Tinta does not collect, store, or transmit any personal data. It has no telemetry, no analytics, no accounts, and no network features.

What Tinta does

Tinta is a markdown and Mermaid diagram viewer for Windows. It reads and renders files you open on your own computer. All processing happens locally on your device.

Data collection

  • Tinta collects no personal information of any kind.
  • Tinta makes no network connections. It contains no telemetry, analytics, crash reporting, or update checks.
  • Clicking a web link in a document opens it in your default browser; Tinta itself never fetches remote content.

Local data

Tinta stores its own settings (theme, zoom level, window position, editor preferences) in a small local configuration file on your device. This file contains no personal information and never leaves your computer.

Microsoft Store

If you install Tinta from the Microsoft Store, Microsoft may provide the developer with aggregate, anonymous statistics (such as install counts) under Microsoft's own privacy statement. Tinta itself reports nothing.

Open source

Tinta's complete source code is publicly available under the MIT license at github.com/oipoistar/tinta — you can verify every claim on this page.

Contact

Questions about this policy? Open an issue at github.com/oipoistar/tinta/issues.

Changes to this policy will be posted on this page with an updated date.
